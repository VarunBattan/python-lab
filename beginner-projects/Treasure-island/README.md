# Treasure Island

A command-line adventure game built in Python where the player makes choices to find hidden treasure.

## Description
The player navigates through a series of decisions such as choosing paths, actions, and doors.  
Each choice affects the outcome of the game, leading to either victory or game over.

## Features
- Multiple decision paths
- Win and lose outcomes based on user choices
- ASCII art displayed at the start of the game

## Concepts Used
- Conditional statements (if / elif / else)
- User input handling
- String comparison
- Program flow control
