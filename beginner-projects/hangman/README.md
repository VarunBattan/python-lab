# Hangman Game

A command-line implementation of the classic Hangman game written in Python.

## Description
The computer randomly selects a word, and the player guesses one letter at a time.  
The player has a limited number of lives. Each incorrect guess reduces a life.  
The game ends when the player guesses the word correctly or runs out of lives.

## Features
- Random word selection
- Letter-by-letter guessing
- Life tracking with visual stages
- Win and lose conditions

## Project Structure
- main.py – core game logic and user interaction
- hangman_words.py – list of possible words
- hangman_art.py – visual stages for remaining lives

## Concepts Used
- Loops
- Conditionals
- Lists
- String manipulation
- Modular programming
