# Password Generator

A Python command-line application that generates a strong random password based on user preferences.

## Description
The program allows the user to specify how many letters, symbols, and numbers they want in their password.  
It then randomly selects characters, shuffles them, and generates a secure password.

## Features
- Custom number of letters, symbols, and numbers
- Random password generation
- Shuffled character order for better security

## Concepts Used
- Random module
- Lists
- Loops
- User input handling
